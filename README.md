# Python Foundations Playground

This repository contains my foundational Python programming work, tracking my learning journey through core language concepts, syntax, and built-in features. 

The primary notebook file (`4.Core_Python.ipynb`) documents my hands-on practice, code experimentation, and implementation of fundamental Python blocks.

## Topics Covered

Here is a breakdown of the core Python concepts implemented in this notebook:

* **String Manipulation & Formatting:** Working with built-in string methods (`upper()`, `strip()`, custom character replacement), escape sequence characters, and formatting patterns.
* **Control Flow & Error Management:** Deep dive into `try-except-finally` code blocks to smoothly handle potential runtime bugs without breaking execution.
* **Custom Exception Handling:** Implementing strict data checking using intentional error raising mechanisms (like `raise ValueError` and `raise NameError`).
* **Interactive Logic:** Capturing user input dynamically and executing conditional matches based on targeted criteria.
